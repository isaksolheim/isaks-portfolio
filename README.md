This website is my personal portfolio.

## Project Status

## Built with
* [React](https://reactjs.org/)
* [Sass](https://sass-lang.com/)

## Installation and Usage
#### Example:
Clone this repository. You will need `node`, and `npm` installed globally on your machine.  

Installation:  

`npm install`  

To Start React App:  

`npm start`  

To Visit App:  

`localhost:3000`
